# CiP-05 – FlightfrontMETAR

FlightfrontMETAR är ett webbaserat verktyg för att tolka och visa METAR-väderdata på ett lättförståeligt sätt för piloter och instruktörer.

---

## Funktioner

- Ange ICAO-kod (t.ex. ESSA) och hämta aktuell METAR
- Klistra in METAR-sträng manuellt
- Automatisk METAR-tolkning
- Visuell presentation med väderikoner
- Rå METAR visas alltid
- Tydliga felmeddelanden vid ogiltig input

---

## Datakälla

METAR hämtas via **CheckWX API**  
`https://api.checkwx.com/metar/<ICAO>`
